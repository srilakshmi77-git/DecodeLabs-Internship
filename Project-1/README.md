# ShopEasy - Responsive E-Commerce Store

## Project Description

ShopEasy is a simple and responsive E-Commerce Store website developed using HTML, CSS, and JavaScript. The project demonstrates the basic concepts of frontend web development, including responsive layouts, product display, search functionality, shopping cart counter, dark mode, and form validation.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Features Implemented

### 1. Responsive Navigation Bar

* Navigation links for Home, Products, About, and Contact.
* Responsive layout for different screen sizes.

### 2. Hero Section

* Welcome banner introducing the store.

### 3. Product Showcase

* Display of multiple products using cards.
* Product image, name, price, and Add to Cart button.

### 4. Search Functionality

* Users can search products using the search bar.
* Products are filtered dynamically using JavaScript.

### 5. Shopping Cart Counter

* Cart count increases whenever a product is added.
* Cart count is displayed in the navigation bar.

### 6. Dark Mode Toggle

* Users can switch between light mode and dark mode.

### 7. Features Section

* Fast Delivery
* Secure Payment
* Customer Support

### 8. Contact Form

* Name, Email, and Message fields.
* Basic form validation using JavaScript.

### 9. Scroll-to-Top Button

* Smoothly scrolls the page back to the top.

### 10. Responsive Design

* Works on desktop, tablet, and mobile devices.

1. Download all project files.
2. Place them in the same folder.
3. Open index.html in a web browser.
4. Start exploring the website.
